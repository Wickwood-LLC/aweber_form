AWeber Webform Module

*** DESCRIPTION ***

The AWeber Web Form module allows users with an AWeber account to add an AWeber web form to a page.
To add a web form in your AWeber account, go to:
http://www.aweber.com/web-forms.htm


*** REQUIREMENTS ***

Drupal 7.x
PHP 5 or greater


*** INSTALLATION ***

To install the module, please follow Drupal's instructions:
http://drupal.org/documentation/install/modules-themes/modules-7

For instructions on usage and setup, please consult our Knowledge Base article:
http://www.aweber.com/faq/questions/598
